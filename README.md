# Moe-Kyaw-Aung-Portfolio V2

Production-ready single-page portfolio landing page for an Android Senior Developer.

## Features
- Hero section with particle.js animated background
- Glassmorphism design
- Responsive Tailwind CSS layout
- About, Projects, and Contact sections
- Single deployable directory structure

## Run
Open `index.html` directly or deploy the folder to Vercel, Netlify, GitHub Pages, or any static host.
